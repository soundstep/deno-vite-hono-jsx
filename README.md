# deno-hono-vite-jsx

I started from this one: <https://github.com/MathurAditya724/honojs-examples/tree/main/deno>

## Requirements

- NodeJS `22.x`
- Deno `2.2.8`
- PNPM `10.7.1`

## Install

```bash
deno install
```

NPM or PNPM can also be used.

## Build the client

Building

```bash
deno task build
```

## Dev server

Run the `vite` dev server with Deno

```bash
deno task dev
```

## Prod server

Run the `deno` prod server

```bash
deno task start
```
